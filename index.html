Hello!

My name's Jamie, I'm an aspiring web developer and musician living in London. I first got into coding when I made an EP of interactive music as a mini computer game. A friend helped me build it, but I loved the process and decided I wanted to learn to code. I’ve been training in HTML, CSS and Javascript for the past year, mostly through online courses / tutorials and my own projects. I’m inspired by interactive music apps and software such as Björk’s Biophilia, and Rebecca Fiebrink's Wekinator.



Projects



In Concert is a web application for an experimental music collaboration project, inspired by open-source principles. Musicians and non-musicians openly share ideas and source materials. Creativity does not exist in a vacuum. This project was built with p5.js and simple Node.js.

Step Music JS is an interactive music composition, which slowely moves through the minor pentatonic scale Circle of 5ths. The user performs the piece by clicking on the screen to trigger a random note in the current scale, and a colour change signifies the scale has modulated. The piece move round in a circle infinitely. This project was built with p5.js.
