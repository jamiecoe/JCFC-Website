Step Music JS

Step Music is an interactive audio/visual display, built around a piece of minimal music. Each user click picks a random note from a 5 note scale. Every 10 clicks, the scale modulates and the colour changes. Each new scale contains only 1 new note and, after 12 modulations, it returns back to the original scale. In this way, the music changes slowly and in a circular motion.
